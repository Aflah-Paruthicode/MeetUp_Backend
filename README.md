 # MeetUp, Server side  

 The robust backend API for the ConnectX social platform, following a decoupled architecture. 

 ### Core Logic
- **User Management:** secure sign-up and login with bcrypt hashing and validator.
- **Request Workflow:** custom logic to handle ignored, interested, accepted and rejected" states between users.
- **Authentication:** jwt (json web tokens) implemented for secure, stateless sessions.
- **Database:** flexible schema design using mongodb and mongoose.

### Tech Stack
- **Runtime:** node.js
- **Framework:** express.js
- **Database:** mongoDB
- **Security:** jwt, bcrypt  

### Under Development
- **Socket.io Integration:** for real time, low latency messaging.
- **Subscription Engine:** logic for tiered user access.
